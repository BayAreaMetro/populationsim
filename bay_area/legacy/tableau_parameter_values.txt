QUICK SETUP STEPS:
------------------

1. Create Parameter:
   - Right-click in Data pane → Create Parameter
   - Name: Select Measure
   - Data type: String
   - Allowable values: List
   - Click Add for each value above (HH, POP, emp_total, etc.)
   - Set Current value: HH
   - Click OK

2. Create Calculated Field:
   - Right-click in Data pane → Create Calculated Field
   - Name: Selected Measure
   - Paste the CASE statement above
   - Click OK

3. Use in visualization:
   - Drag "Selected Measure" to Color shelf
   - Right-click parameter "Select Measure" → Show Parameter
   - Dropdown will appear at top-right of worksheet
